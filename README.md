## Portfolio — Muhammad Ariel Ariadi

Portfolio website personal yang dibangun dari scratch menggunakan HTML, CSS, dan JavaScript.

**Live:** [Coming soon]

### Tech Stack

- HTML5 (Semantic)
- CSS3 (Custom Properties, Flexbox, Grid)
- JavaScript (Vanilla, IntersectionObserver)

### Features

- Mobile-first responsive design
- Scroll-triggered animations
- Glassmorphism navigation
- WCAG accessibility compliant
- Clean & semantic HTML structure

### Structure

```
├── index.html       # Portfolio website
├── style.css        # Design system & styles
├── script.js        # Interactivity & animations
├── assets/          # Screenshots & images
└── plan/            # Process documentation
    ├── 01-brainstorm.md
    ├── 02-details.md
    ├── 03-execution.md
    └── 04-results.md
```

### Run Locally

Buka `index.html` di browser, atau gunakan Live Server extension di VS Code.
